# lightshuttle

> Lightweight dev orchestrator for polyglot teams: typed `docker-compose`
> successor with built-in dashboard, OpenTelemetry and production export.

LightShuttle is a developer-time orchestrator written in Rust. This crate
hosts the `lightshuttle` binary; the manifest model and runtime live in
the sibling crates `lightshuttle-manifest` and `lightshuttle-runtime`.

## Install

```sh
cargo install lightshuttle
```

This release is pre-alpha and intentionally does nothing useful yet.
The name is published to reserve it on crates.io; usable functionality
ships with v0.1.0.
