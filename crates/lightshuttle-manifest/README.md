# lightshuttle-manifest

Manifest types, parser, interpolation and JSON Schema generation for
[LightShuttle](https://github.com/nubster-opensources/lightshuttle), a
lightweight developer-time orchestrator for polyglot teams.

This crate is part of the LightShuttle workspace. See the main
repository for documentation, roadmap and contribution guidelines.

## Status

Pre-alpha. Not usable yet. This release reserves the name on crates.io;
the public API will stabilise with LightShuttle v0.1.0.

The manifest specification this crate targets lives at
[`docs/spec/manifest-v0.md`](https://github.com/nubster-opensources/lightshuttle/blob/main/docs/spec/manifest-v0.md).
