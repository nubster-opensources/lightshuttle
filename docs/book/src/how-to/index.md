# How-to guides

How-to guides are task-oriented: they assume you already know the basics and
want to accomplish something specific. Unlike a tutorial, a how-to guide solves
one real problem and does not stop to explain every concept along the way.

Guides for secrets and environment management, dependencies, networking,
observability and troubleshooting are being added to this section. In the
meantime, the [getting started tutorial](../tutorials/getting-started.md) covers
the most common tasks end to end.
